info About Power Users

Power users have elevated privileges compared to regular customers. They can:

  • Access event management features
  • View detailed sales analytics
  • Create and modify events
  • Manage event statuses

Note: Power users cannot manage other users or access full admin features. Only ADMIN users can manage users and create power users.
